# Typing Speed Test

A typing speed test web app based on a Figma design.  
The project focuses on clean layout, UI structure, and future JavaScript integration.

## Features

- Statistics section (WPM, accuracy, time)
- Difficulty selection (Easy / Medium / Hard)
- Mode selection (Timed / Passage)
- Personal best placeholder
- Typing area with blur and start overlay (not started state)

## Tech Stack

- HTML5
- CSS3 (Flexbox)
- JavaScript (planned)

  ## Project Status

- Home page layout completed: done
- Not started state implemented: done
- Typing logic: in progress
- Statistics calculation (WPM, accuracy): in progress
